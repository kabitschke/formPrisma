#Este formulário faz as validação dos campos, utilizando zod e react-hook-form.

## Tecnologias utilizadas
- React
- TypeScript    
- Zod
- React Hook Form

## Como rodar o projeto
1. Clone o repositório git clone https://github.com/kabitschke/formPrisma.git
2. Instale as dependências com `npm install`
3. Inicie o projeto com `npm run dev`
4. Acesse `http://localhost:3000` no seu navegador

## Funcionalidades
- Validação de campos utilizando Zod
- Gerenciamento de formulários com React Hook Form
